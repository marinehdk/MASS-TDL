Architecture Principles (v3.0):
1. Adjacent-layer nominal: Normal ops each layer talks only to neighbors. Exceptions: Reflex Arc (Sensor→L5), Hardware Override (Physical→Actuator).
2. Perception independence: Multimodal fusion feeds Decision stack + Checker + Reflex + Nav Filter.
3. Doer-Checker separation: AI decision (Doer) ∥ deterministic verifier (Checker). Veto + nearest compliant alternative.v2.0
4. IT/OT hard isolation: DMZ + Data Diode. IACS UR E26/E27.v2.0
5. Human override supremacy: Physical relay > all software. Zero-software cutoff.v2.0
6. Dual-redundant computing: Two independent nodes, hot standby, auto failover. No single-point failure.v3.0
7. Sensor redundancy & voting: Critical sensors 2×~3×. Triple: median vote. Dual: difference test. Auto-exclude degraded source.v3.0
8. Navigation Filter as single source of truth: All layers consume own-ship state from Nav Filter EKF, not raw GPS. DR fallback on GNSS loss.v3.0
9. Multi-ship-type parametric: Ship dynamics, thruster config, sensor layout via YAML. Switch ship type without code change.v3.0
10. Graceful degradation: Sensor loss → reduce autonomy. Thruster loss → capability assessment → reduce speed/heading authority. Compute loss → hot standby takeover.
