# D1.2 CI/CD Pipeline — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Harden the MASS-L3 GitLab CI 5-stage pipeline — pin all Docker image versions via SHA256 + apt version locks, eliminate all 8 packages' clang-tidy violations (PATH-S for M1/M7, PATH-D for others), extend PATH-S Doer→Checker detection to C++, and convert all remaining soft-fail CI jobs to hard-fail.

**Architecture:** Two independent work streams run in parallel: (A) Docker version locking (`audit-versions.sh` → `version-lock.txt` → Dockerfile ARG pins → build verification), and (B) YAML patch (`path_s_dry_run` C++ extension + 3 soft-fail removals). Stream A gates all clang-tidy fix work (Waves 4-5). Stream B is mergeable independently. Final task T13 gates on all prior tasks via a full-pipeline green.

**Tech Stack:** Bash, Docker BuildKit, `run-clang-tidy-20`, `colcon`, GitLab CI YAML

- [ ] **Step A-1: Create `audit-versions.sh`**

```bash
cat > tools/docker/audit-versions.sh << 'SCRIPT'
#!/usr/bin/env bash
# Run from repo root on a host with Docker daemon access.
# Requires ros:jazzy-ros-base-jammy to be already pulled.
# Output: version-lock.txt entries printed to stdout.
set -euo pipefail

IMAGE="ros:jazzy-ros-base-jammy"

echo "# Auto-generated by audit-versions.sh — commit after review"
echo "# Update flow: run this script -> git commit version-lock.txt -> docker build"
echo ""

DIGEST=$(docker inspect "${IMAGE}" \
           --format='{{index .RepoDigests 0}}' 2>/dev/null | cut -d@ -f2 || echo "UNKNOWN")
echo "ROS_BASE_DIGEST=${DIGEST}"

docker run --rm "${IMAGE}" bash -c "
  export DEBIAN_FRONTEND=noninteractive
  apt-get update -qq 2>/dev/null
  for pkg in gcc-14 clang-tidy-20 cmake cppcheck python3.10; do
    ver=\$(apt-cache show \"\$pkg\" 2>/dev/null | grep '^Version:' | head -1 | awk '{print \$2}')
    varname=\$(echo \"\${pkg^^}_VER\" | tr '-.' '__')
    echo \"\${varname}=\${ver}\"
  done
"
echo "# colcon-common-extensions: no PyPI release tag — known exception, not pinned"
SCRIPT
chmod +x tools/docker/audit-versions.sh
```
