## 1. 设计定位

仿真控制工具条是 Captain HMI 和运行报告页面的统一组件，提供仿真运行期间的时间控制、速率调整、场景回放等功能。

### 1.1 设计原则
- **视频播放器风格**：符合用户操作习惯
- **固定底部**：始终可见，方便随时控制
- **响应式**：适配不同屏幕尺寸

## 3. 控制按钮区

### 3.1 按钮列表

| 按钮 | 图标 | 快捷键 | 状态 |
|------|------|--------|------|
| **播放** | ▶ | `Space` | 暂停/停止后可点击 |
| **暂停** | ⏸ | `Space` | 运行中可点击 |
| **停止** | ⏹ | `Escape` | 运行/暂停中可点击 |
| **重置** | ⏮ | `R` | 停止后可点击 |

### 3.2 按钮样式

```
┌─────────┐
│    ▶    │  ← 40×40px
│  Play   │  ← 下方文字
└─────────┘
```

**状态样式**：
| 状态 | 按钮颜色 | 说明 |
|------|---------|------|
| 可用 | 主题蓝 | 默认颜色 |
| 不可用 | 灰色 50% 透明度 | 禁用状态 |
| 激活 | 主题绿 | 当前激活按钮（如播放中暂停高亮）|

### 3.3 快捷键绑定

| 快捷键 | 操作 |
|--------|------|
| `Space` | 播放/暂停切换 |
| `Escape` | 停止 |
| `R` | 重置 |
| `←` | 后退 10 秒 |
| `→` | 前进 10 秒 |
| `1` | 1x 速率 |
| `2` | 2x 速率 |
| `5` | 5x 速率 |

## 4. 时间轴 Scrubber

### 4.1 组件设计

```
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
0:00      2:00      4:00      6:00      8:00     10:00
          │
          ● ← 当前位置（可拖动）

已播放区域：███████░░░░░░░░░░░░░░░░░░░░░░
```

### 4.2 时间显示

| 显示项 | 格式 | 位置 |
|--------|------|------|
| 当前时间 | `MM:SS` 或 `H:MM:SS` | 时间轴左下方 |
| 总时长 | `MM:SS` 或 `H:MM:SS` | 时间轴右下方 |

**示例**：
```
3:42 / 10:00
```

### 4.3 拖动交互

| 交互 | 行为 |
|------|------|
| 点击轨道 | 跳转到点击位置 |
| 拖动滑块 | 实时更新地图和指标 |
| 拖动时按住 | 显示精确时间提示 |

### 4.4 事件标注

关键事件在时间轴上显示为小红点：

```
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
0:00      2:00      4:00      6:00      8:00     10:00
                ●            ●                ●
              ToR         COLREG            CPA
```

**悬停显示**：
```
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
0:00      2:00      4:00      6:00      8:00     10:00
                ●
              4:00
            COLREG R14
```

## 6. 扩展功能区

### 6.1 注入故障按钮

**按钮**：`注入故障 ▼`

**点击展开菜单**：
```
┌────────────────────────────────────┐
│ 注入故障                           │
├────────────────────────────────────┤
│ ○ AIS 掉线                         │
│ ○ Radar 噪声尖峰                   │
│ ○ 环境扰动突变                      │
│ ○ 目标船异常行为                    │
│ ○ 传感器延迟                        │
│ ○ 自定义...                        │
├────────────────────────────────────┤
│  取消            确定              │
└────────────────────────────────────┘
```

**故障类型**：

| 故障类型 | 参数 | 说明 |
|---------|------|------|
| AIS 掉线 | 持续时间 | AIS 消息丢失 |
| Radar 噪声尖峰 | 持续时间 | 检测概率降低 |
| 环境扰动突变 | 风/流变化量 | 突然变化 |
| 目标船异常 | 异常类型 | 突然转向/加速 |
| 传感器延迟 | 延迟时间 | 所有传感器延迟 |

### 6.2 重置按钮

**按钮**：`重置`

**点击后行为**：
1. 确认弹窗
2. 重置时间到 0
3. 清除所有运行时状态
4. 重新加载场景配置

**确认弹窗**：
```
┌────────────────────────────────────────┐
│ 确认重置仿真？                         │
│                                        │
│ 所有运行时数据将被清除。                │
│ MCAP 录制将保存为新文件。              │
│                                        │
│        [取消]        [确认重置]        │
└────────────────────────────────────────┘
```

## 7. 简化版本（运行报告中）

运行报告页面的工具条为简化版本，不包含故障注入：

```
┌──────────────────────────────────────────────────────────────────────────────────────┐
│  ⏮ │ ▶ │ ⏭  ──●──────────────────────  3:42/10:00 ──  [0.5x] [1x] [2x] [4x]      │
│  后退   播放   前进                                    播放速度选择                   │
└──────────────────────────────────────────────────────────────────────────────────────┘
```

### 7.1 播放控制

| 按钮 | 图标 | 功能 |
|------|------|------|
| 后退 | ⏮ | 后退 10 秒 |
| 播放/暂停 | ▶/⏸ | 切换播放状态 |
| 前进 | ⏭ | 前进 10 秒 |

### 7.2 速度选择

使用按钮组而非下拉：

```
[0.5x] [1x] [2x] [4x]
```

## 8. 状态管理

```typescript
// useSimulationControlStore
interface SimulationControlStore {
  // 状态
  isPlaying: boolean;
  isPaused: boolean;
  simRate: number;
  currentTime: number;
  duration: number;
  
  // 故障
  activeFaults: Fault[];
  availableFaults: FaultDefinition[];
  
  // 操作
  play: () => Promise<void>;
  pause: () => Promise<void>;
  stop: () => Promise<void>;
  reset: () => Promise<void>;
  seekTo: (time: number) => Promise<void>;
  setRate: (rate: number) => Promise<void>;
  injectFault: (fault: FaultDefinition) => Promise<void>;
}
```
